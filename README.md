## 📖 About

**Strategy Tracker** is a professional-grade backtesting and performance tracking tool designed for individual traders.

Unlike complex quantitative platforms that require heavy setup, this tool focuses on one core purpose: **Fairly comparing different trading strategies using real-world data to find the "truth" in performance.**

It is built upon **Cybernetics (控制论)** principles: establishing a **negative feedback loop** where strategy performance feeds back into strategy selection, creating an adaptive trading system.

### 🧠 Core Philosophy

1.  **First Principles**: Strip away marketing jargon and focus on the input-output relationship (Signal → Return).
2.  **Fair Comparison**: Strict control over initial capital, slippage, and commission rates to ensure "apples-to-apples" comparison.
3.  **Visual Insight**: Generating beautiful ECharts dashboards to visualize equity curves and drawdowns instantly.

---

## ✨ Features

- 🚀 **Real-World Data**: Seamless integration with `akshare` and local stock APIs to fetch real A-share data.
- ⚖️ **Fair Benchmarking**: Run multiple strategies (e.g., CATS-A vs. Moving Average vs. Buy & Hold) simultaneously.
- 📉 **Key Metrics**: Auto-calculates Total Return, Max Drawdown, Sharpe Ratio, and Win Rate.
- 🎨 **ECharts Visualization**: Generates a standalone HTML report with interactive equity curves and data tables.
- 🔌 **Extensible API**: Register any custom strategy function easily.

## 🚀 Quick Start

### 1. Installation

Clone the repository:

```bash
git clone https://github.com/GroAries/strategy-tracker.git
cd strategy-tracker
```

Dependencies:
- Python 3.10+
- `akshare` (for data)
- `pandas`, `numpy`

### 2. Usage (Python API)

```python
from strategy_tracker.tracker import StrategyTracker

# 1. Initialize
tracker = StrategyTracker(initial_capital=100000)

# 2. Register Strategies
tracker.add_strategy("Buy & Hold", lambda df: [1] + [0]*(len(df)-1))
tracker.add_strategy("MA Cross", my_ma_cross_function)

# 3. Fetch Data
df = tracker.fetch_data("600519", "20250101", "20251231")

# 4. Run & Report
results = [tracker.run_backtest(name, df) for name in tracker.strategies]
html = tracker.generate_report(results, "Kweichow Moutai (600519)")

with open("report.html", "w") as f:
    f.write(html)
```

### 3. Run Demo

See the working example in the `examples/` folder:

```bash
cd examples
python3 run_real_demo.py
# Opens real_data_report.html
```
